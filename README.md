# 🏀 Team Hub Pro v2 — Sports News Aggregator

A fully automated, ESPN-style sports hub that aggregates the latest news, videos, insider coverage, rankings, schedule, and NIL data — all auto-updating every 30 minutes.

---

## 🚀 Features

- 📰 **Headline Ticker** – Rotating top news headlines  
- 🏆 **Featured Story Hero** – Most important article front and center  
- 📺 **Video Carousel** – YouTube highlights, insider breakdowns  
- 📝 **Latest Headlines** – National + local + official coverage  
- 📣 **Insider Coverage** – Beat writers, premium blogs, recruiting news  
- 📊 **Rankings Sidebar** – AP Poll & KenPom auto-updated  
- 💰 **NIL Leaderboard** – Top player valuations  
- 📅 **Schedule Sidebar** – Upcoming games  
- 🔁 **Auto-Updating Feed** – Refreshes every 30 minutes via GitHub Action
