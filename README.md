# Purdue MBB Hub — Live Build
Deploys automatically via `.github/workflows/collect.yml`.

## How to use
1. Upload all files (replace old ones).
2. Go to **Settings → Pages → Build from GitHub Actions**.
3. Run **Actions → Build and Deploy Purdue MBB Hub → Run workflow**.
4. Visit [https://jasonab74-ctrl.github.io/purdue-mbb-bot/](https://jasonab74-ctrl.github.io/purdue-mbb-bot/).
