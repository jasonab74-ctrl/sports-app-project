# 🏀 Sports App Project — One-Stop Team Hub

### 📌 Overview

The **Sports App Project** is a turn-key, mobile-first web app that aggregates everything a fan needs to know about a single sports team — news, videos, rankings, schedule, roster, NIL, and insider content — into one auto-updating hub.

This repo powers sites like:

- [Purdue Men’s Basketball Hub](https://jasonab74-ctrl.github.io/sports-app-project/)
- Future teams: NFL, NBA, MLB, NCAA Football, etc.

The goal: **No more hunting across 8 sites.** Fans open one page and see *everything*.

---

## 🚀 Features

✅ **Auto-Updating Feeds** — pulls the latest 20+ articles and highlights from trusted news and video sources  
✅ **Video Highlights** — auto-populates from YouTube channels (e.g., Field of 68, official team channels)  
✅ **Rankings & Analytics** — includes AP Poll, KenPom, and team stats  
✅ **NIL Leaderboard** — track top-valued players and NIL deals  
✅ **Schedule & Roster** — full season schedule and player info  
✅ **Insider Coverage** — direct access to beat writers, podcasts, and premium sources  
✅ **Mobile-First UX** — designed for iOS/Android as a progressive web experience  
✅ **Team-Themed Branding** — color palette, logo, and typography adapt to the selected team  
✅ **Auto-Updating GitHub Action** — refreshes data every 30 minutes without manual input
